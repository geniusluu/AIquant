海龟交易法则核心思想

什么是海龟交易法则？

海龟交易法则（Turtle Trading）由理查德·丹尼斯于1983年创立，是量化交易史上最著名的趋势跟踪系统之一。丹尼斯与合伙人打赌「交易能力可以后天培养」，招募了一批「海龟」学员，用同一套规则交易，4年间赚了超过1.7亿美元。

海龟法则的核心不是某个单一指标，而是一套从入场到退出的完整风险控制链。传统双均线策略满仓进出、无止损、无仓位管理，回撤不可控。海龟增强版通过四重机制将单笔最大亏损锁定在总资金的1%~3%。

四大核心机制

  1. 波动率定仓（ATR Position Sizing）：根据ATR决定仓位大小，波动大少买、波动小多买，使每笔交易承担的风险金额大致相等。单笔最大亏损 = 总资金 × 风险比例（1%）。
  2. 通道突破过滤（Donchian Channel）：价格突破过去N日最高/最低点作为信号参考。本策略以均线金叉为主信号，10日低点通道作为退出信号。
  3. 金字塔加仓（Pyramid Scaling）：入场后每上涨0.3个ATR追加一个头寸单位，最多3个单位。加仓后止损线随均价上移，实现「赢时加仓」的正反馈。
  4. ATR动态止损（ATR Stop-Loss）：持仓均价 - 3×ATR作为硬止损线。ATR随波动率动态调整，波动大时止损线自动放宽，避免被正常波动震出局。

关键优势

在保持趋势跟踪能力的同时显著降低极端下行风险。最大回撤从基线/持有的-22%~41%降低到-16%~20%，夏普比率在部分标的上超越基线。

核心指标解释

一、唐奇安通道（Donchian Channel）

唐奇安通道由理查德·唐奇安发明，是最早的通道类技术指标之一。计算方式为：取过去N日的最高价和最低价，形成上下两条边界线。

  • 通道上轨 = 过去20日最高价（DC_high），判断价格是否突破近期高点
  • 通道下轨 = 过去10日最低价（DC_low），作为趋势止盈退出信号
  • 通道宽度 = DC_high - DC_low，反映市场波动范围（窄=震荡，宽=趋势）

通道退出比均线死叉反应更快：死叉需要短期均线下穿长期均线（滞后），而通道退出只需价格跌破近期低点（更敏锐），在趋势末端能保住更多利润。

二、平均真实波幅（ATR, Average True Range）

ATR由威尔斯·威尔德发明，衡量市场波动率的指标。计算步骤：

  1. 真实波幅(TR) = max(当日最高-最低, |当日最高-昨收|, |当日最低-昨收|)，取三者最大值，考虑了跳空缺口
  2. ATR(N) = 过去N日TR的移动平均值（本策略N=20）

ATR的关键特性：

  • 自适应性：波动大时ATR自动增大，波动小时自动缩小
  • 绝对值含义：ATR=0.85表示该股日均波动约0.85元
  • 三重用途：仓位计算（ATR大→少买）、止损设置（ATR大→止损放宽）、加仓间距（ATR大→间距更远）

三、止损条件设计 — 三层退出机制

第一层：ATR硬止损（最高优先级）

  • 止损线 = 持仓均价 - 3 × ATR(20)，当日最低价触及止损线 → 次日开盘全部清仓
  • 3倍ATR = 给价格3个平均日波动率的「呼吸空间」，避免被正常回调震出局
  • 加仓后止损线随均价上移，锁定已有利润

第二层：通道退出（趋势止盈）

  • 收盘价跌破过去10日最低价 → 次日开盘清仓，反应比死叉更快

第三层：均线死叉退出

  • SMA(5)下穿LMA(20) → 次日开盘清仓，作为兜底信号

三层退出优先级：止损 > 通道退出 > 死叉。同一日只执行最高优先级的动作。

参数对策略收益的影响

通过864组参数扫描和双标的验证，发现以下规律：

1. 止损倍数（影响最大）

紫金矿业从2.0×ATR放宽到3.0×ATR，收益从-9.7%翻正至+19.3%——这是决定策略盈亏的关键参数。止损太紧=频繁假止损=亏损；止损太宽=单笔亏损过大。最优值通常在2.5~3.5倍。

2. 风险比例（收益与风险的杠杆）

提高风险比例直接放大仓位和收益，但回撤同步扩大。紫金矿业1%最优（夏普0.354），3%时回撤恶化到-32.7%。兆易创新3%仍最优（夏普3.0），因趋势足够强。

3. 加仓间距（趋势段利润放大器）

0.3比0.5更好，更早加仓能在趋势段建立更重仓位。但间距太小会导致加仓过快、风险集中。

4. ATR周期（灵敏度vs稳定性）

20日最优。10日太敏感（ATR波动大导致仓位频繁变化），20日足够平滑。

核心规律：参数最优值与标的波动率强相关。低波动标的（紫金）宜宽止损+低风险比例；高波动标的（兆易）可适当提高风险比例，止损倍数影响较小。

海龟法则的适用场景

最适合的场景

  • 高波动+强趋势标的：如兆易创新（11倍涨幅），海龟168.9%收益、回撤仅-16.7%，风控优势显著
  • 震荡+趋势交替行情：ATR止损能在震荡段及时止血，通道退出能在趋势末端保住利润
  • 多标的组合：不同波动率的标的搭配，ATR自动平衡各标的的风险贡献

不太适合的场景

  • 纯单边慢牛行情：如紫金矿业这段，价格稳步上涨少回撤，满仓持有最优，海龟轻仓反而踏空
  • 极低波动标的：ATR太小导致单位股数过多，单笔风险可能超过预期
  • 高频交易：海龟是趋势跟踪系统，持仓周期通常数周到数月

本质判断标准

如果标的的「趋势段收益 / 震荡段亏损」比值大于1，海龟策略就能盈利；如果市场以震荡为主（假信号多、止损频繁），海龟会持续亏损。

使用心得与改进方向
  1. 风险比例是「收益-风险」的总开关：1%保守、2%平衡、3%激进。建议从1%起步，确认策略逻辑有效后逐步提高。
  2. 止损倍数决定生死：宁可宽一些（3倍ATR）也不要太紧（2倍以下）。宽止损的代价是单笔亏损金额更大，但换来的是不被正常波动震出局。
  3. 加仓是双刃剑：趋势延续时加仓增厚利润，趋势反转时加仓放大亏损。必须配合止损使用，且加仓后止损线要随均价上移。
  4. 通道退出优于纯死叉：10日低点退出比死叉快2~5天，在趋势末端能多保住3~8%的利润。
  5. 不要对单一标的过度优化参数：864组扫描找到的「最优参数」可能只是过拟合。更好的做法是在多个标的上测试参数鲁棒性。

改进方向

  • (1) 加入ADX指标过滤：ADX>25才执行加仓，避免震荡市中盲目加仓
  • (2) 动态风险比例：根据ATR/价格比率自动调整风险比例（波动大时降低）
  • (3) 多标的风险预算：总风险不超过6%，单标的不超过2%，实现组合层面风控
  • (4) 分批止盈：达到1倍ATR利润时减半仓，剩余仓位用通道退出跟踪趋势
